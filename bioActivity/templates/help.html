How to Use:

  1. Choose a file to upload by clicking the Choose File button.
  2. Select the file containing a list of SMILES strings that you wish to predict. Ensure the file is in .csv, .xlsx, or .txt format. Here's an example of a SMILES string for input:
    3. CCOP(=S)(OCC)SCSP(=S)(OCC)OCC CC(C)OC1=CC=CC=C1OC(=O)NC CNC(=O)CSP(=S)(OC)OC CCOP(=S)(OCC)OC1=CC=C(C=C1)[N+](=O)[O-] COP(=O)(C(C(Cl)(Cl)Cl)O)OC COP(=S)(OC)SCN1C(=O)C2=CC=CC=C2N=N1 CCOP(=S)(OCC)OC1=NC(=C(C=C1Cl)Cl)Cl COC1=C(C=C2C(=C1)CC(C2=O)CC3CCN(CC3)CC4=CC=CC=C4)OC CC1(CC2=C(O1)C(=CC=C2)OC(=O)NC)C CC(=O)NP(=O)(OC)SC
  3. Next, click the Predict button to obtain the predictions.
  4. Once the results page is displayed, you can download the prediction results by clicking the Download as CSV button.
  5. Analyze each molecule by clicking the Analyze button.
  6. On the Analysis page, you can download the PDB file of a molecule by clicking the Download PDB button below the molecule visualization.
